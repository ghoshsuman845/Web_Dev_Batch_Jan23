Cascading and Specificity

  1. Position and Order or appearance: the browser applies the styles appearing at the end.
  2. Origin:
    • User Agent styles: these styles are the once applied by browser to your html element by default
    • Local user styles: these styles are applied by the operating system
    • Authored CSS: the css file that you have written
    • Authored CSS !important
    • Local user styles !important
    • User Agent styles !important
  3. Specificity: The algo which tells which selector has more priority based on the specificity score
  4. !important : Highest priority
