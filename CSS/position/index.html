CSS Positions

  1. Static: The original or default position of an element.
  2. Relative:Element is positioned relative to it's normal/original position. Same as static positioning, just that here we can move an element by adding offset values. Element stays in normal document flow.
  3. Absolute: Element is positioned relative to it's parent. If parent is not present then element is positioned relative to the main document. Element is removed from the normal document flow, no space is created for that element.
  4. Fixed: Element is positioned relative to the viewport. Element does not move and stays in the same position even on scrolling. Element is removed from normal flow.
  5. Sticky: Elelemnt is positioned according to the normal document flow. Till the element hits the top, it has normal position, after it touches the top then it's position becomes fixed.
